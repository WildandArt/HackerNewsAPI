\end{DoxyItemize}\hypertarget{md_HELP_autotoc_md3}{}\doxysubsection{Maven Parent overrides}\label{md_HELP_autotoc_md3}
Due to Maven\textquotesingle{}s design, elements are inherited from the parent POM to the project POM. While most of the inheritance is fine, it also inherits unwanted elements like {\ttfamily $<$license$>$} and {\ttfamily $<$developers$>$} from the parent. To prevent this, the project POM contains empty overrides for these elements. If you manually switch to a different parent and actually want the inheritance, you need to remove those overrides.
